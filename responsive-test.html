Responsive Design Test

How to Test Responsiveness:

  • Desktop (1200px+): Full layout with side-by-side hero content
  • Laptop (992px-1199px): Slightly smaller profile image, adjusted spacing
  • Tablet (768px-991px): Stacked layout, image above text, centered content
  • Mobile Landscape (576px-767px): Compact layout, smaller profile image, stacked buttons
  • Mobile Portrait (up to 575px): Minimal spacing, small profile image, vertical button layout
  • Extra Small (up to 400px): Ultra-compact design for very small screens
